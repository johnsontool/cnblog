使用Flask和Nginx构建后端API系统
发布于：2024-03-27 11:04:26 标签：/ python / flask / nginx / api / 访问：

本文主要介绍基于Flask和Nginx构建简单的API系统。作为演示，该系统只有一个后端API，接收来自前端的参数并返回生成的随机数。整个部署环境是Ubuntu 18.04操作系统。

Flask是Python下的轻量级Web应用程序框架，主要优点：

  • 轻量级且足够简单易用
  • 基于Python，易于上手

使用Nginx与Flask的基本架构图如下：

  • Nginx作为Web服务器，接收用户请求并反向代理到Gunicorn服务器
  • Gunicorn是WSGI（Web服务器网关接口）HTTP服务器，运行Python Web应用程序
  • Supervisor是一个进程管理工具，可以监控Gunicorn的运行，如果意外关闭可以自动重启
  • Flask App是我们开发的应用服务
